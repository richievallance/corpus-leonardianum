## METHODOLOGY

**BNE File Numbering System:**
- BNE digital archive counts DOWN from f.0r
- BNE file 5958 = f.0r (opening folio)
- Therefore: BNE file N = f.(5958-N)r or v

**Example:**
- BNE 5869 = f.0r + 89 folios = approximately f.44v-45r range
- BNE 5955 = f.0r + 3 folios = f.1v-2r range

**Note:** BNE numbering is approximate due to complex recto/verso sequencing and multi-column subdivisions (A/B/C sections per folio).
