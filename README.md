## Purpose

Strict manuscript infrastructure repository. Contains IIIF manifests, codicological data, folio sequence mappings, reconstruction audit logs, and BNE transcript correlation matrices for Codex Madrid I and II.

## Key Facts

- BNE numbering is inverse: file 5958 = f.0r; decreasing BNE numbers = increasing folio numbers
- Folios f.36v–f.43r are physically missing from Madrid Codex I
- BNE thematic categories do not follow physical folio sequence
